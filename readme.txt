HOW TO RUN

Folder Structure:
soft_sys2/
    Problem_1/
        encryptor.py
        raw_text.txt
    Problem_2/
        evaluator.py
        sample_input.txt


Open terminal inside the soft_sys2 folder or each subfolder, then run:

Q1:
python Problem_1/encryptor.py
(it will ask for shift1 and shift2 then run automatically and save files)

Q2:
python Problem_2/evaluator.py Problem_2/sample_input.txt
(output saves to Problem_2/output.txt automatically)
